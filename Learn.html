Basic HTML

First you'll need a header, to make one use <h1 >..</h1 >. There are six types of headers, each being smaller than the last. The largest is <h1> and the smallest is <h6>.

Then write some text. To make a paragraph use <p>...</p>. For Bold use <strong>...</strong> and italics use <em>...<em>. Here's how you make a link: <a href="example.com">Click Me</a>. To write a list, first type <ul>, then on the next line type <li>...<li> to add an item. Now, to end the list type </ul>.
